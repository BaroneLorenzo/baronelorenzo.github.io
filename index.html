PHP

PHP è un linguaggio di scripting generico particolarmente adatto allo sviluppo web. È stato originariamente creato dal programmatore danese-canadese Rasmus Lerdorf nel 1994; l'implementazione di riferimento PHP è ora prodotta da The PHP Group. PHP originariamente stava per Personal Home Page, ma ora sta per l'inizialismo ricorsivo PHP: Hypertext Preprocessor. Il codice PHP viene solitamente elaborato su un server web da un interprete PHP implementato come modulo, demone o eseguibile CGI (Common Gateway Interface). Su un server web, il risultato del codice PHP interpretato ed eseguito, che può essere qualsiasi tipo di dato, come HTML generato o dati di immagini binarie, formerebbe l'intera o parte di una risposta HTTP. Esistono vari sistemi di template web, sistemi di gestione dei contenuti web e framework web che possono essere impiegati per orchestrare o facilitare la generazione di quella risposta. Inoltre, PHP può essere utilizzato per molte attività di programmazione al di fuori del contesto web, come applicazioni grafiche autonome e controllo robotico dei droni. Il codice PHP arbitrario può anche essere interpretato ed eseguito tramite l'interfaccia della riga di comando (CLI).

HTML

Hypertext Markup Language (HTML) è il linguaggio di markup standard per i documenti progettati per essere visualizzati in un browser web. Può essere assistito da tecnologie come Cascading Style Sheets (CSS) e linguaggi di scripting come JavaScript. I browser Web ricevono i documenti HTML da un server Web o dalla memoria locale ed eseguono il rendering dei documenti in pagine Web multimediali. HTML descrive la struttura di una pagina web semanticamente e originariamente includeva segnali per l'aspetto del documento.

Cascading Style Sheets

Cascading Style Sheets (CSS) è un linguaggio per fogli di stile utilizzato per descrivere la presentazione di un documento scritto in un linguaggio di markup come l'HTML. CSS è una tecnologia fondamentale del World Wide Web, insieme a HTML e JavaScript. CSS è progettato per consentire la separazione di presentazione e contenuto, inclusi layout, colori e caratteri. Questa separazione può migliorare l'accessibilità del contenuto, fornire maggiore flessibilità e controllo nella specifica delle caratteristiche di presentazione, consentire a più pagine Web di condividere la formattazione specificando il CSS pertinente in un file .css separato che riduce la complessità e la ripetizione nel contenuto strutturale oltre a consentire il file .css da memorizzare nella cache per migliorare la velocità di caricamento della pagina tra le pagine che condividono il file e la sua formattazione.

Java Script

JavaScript, spesso abbreviato in JS, è un linguaggio di programmazione conforme alla specifica ECMAScript. JavaScript è di alto livello, spesso compilato just-in-time e multi-paradigma. Ha la sintassi delle parentesi graffe, la digitazione dinamica, l'orientamento agli oggetti basato su prototipi e funzioni di prima classe. Insieme a HTML e CSS, JavaScript è una delle tecnologie principali del World Wide Web. JavaScript abilita pagine web interattive ed è una parte essenziale delle applicazioni web. La stragrande maggioranza dei siti Web lo utilizza per il comportamento della pagina lato client e tutti i principali browser Web dispongono di un motore JavaScript dedicato per eseguirlo. Essendo un linguaggio multi-paradigma, JavaScript supporta stili di programmazione basati su eventi, funzionali e imperativi. Dispone di API (Application Programming Interface) per lavorare con testo, date, espressioni regolari, strutture dati standard e DOM (Document Object Model). Tuttavia, il linguaggio stesso non include input / output (I / O), ad esempio funzionalità di rete, archiviazione o grafica, poiché l'ambiente host (di solito un browser Web) fornisce tali API.

MySql

MySQL è un sistema di gestione di database relazionali (RDBMS) open source. Il suo nome è una combinazione di "My", il nome della figlia del co-fondatore Michael Widenius, e "SQL", l'abbreviazione di Structured Query Language. Un database relazionale organizza i dati in una o più tabelle di dati in cui i tipi di dati possono essere correlati tra loro; queste relazioni aiutano a strutturare i dati. SQL è un linguaggio utilizzato dai programmatori per creare, modificare ed estrarre dati dal database relazionale, nonché per controllare l'accesso degli utenti al database. Oltre ai database relazionali e SQL, un RDBMS come MySQL funziona con un sistema operativo per implementare un database relazionale nel sistema di archiviazione di un computer, gestisce gli utenti, consente l'accesso alla rete e facilita il test dell'integrità del database e la creazione di backup.

Java

Java è un linguaggio di programmazione basato su classi e orientato agli oggetti progettato per avere il minor numero possibile di dipendenze di implementazione. È un linguaggio di programmazione generico progettato per consentire agli sviluppatori di applicazioni di scrivere una volta, eseguire ovunque (WORA), il che significa che il codice Java compilato può essere eseguito su tutte le piattaforme che supportano Java senza necessità di ricompilazione. Le applicazioni Java vengono in genere compilate in bytecode che può essere eseguito su qualsiasi macchina virtuale Java (JVM) indipendentemente dall'architettura del computer sottostante. La sintassi di Java è simile a C e C ++, ma ha meno funzionalità di basso livello rispetto a entrambe. A partire dal 2019, Java era uno dei linguaggi di programmazione più popolari in uso secondo GitHub, in particolare per le applicazioni web client-server, con 9 milioni di sviluppatori segnalati.

Python

Python è un linguaggio di programmazione interpretato, di alto livello e generico. Creato da Guido van Rossum e rilasciato per la prima volta nel 1991, la filosofia di progettazione di Python enfatizza la leggibilità del codice con il suo notevole uso di spazi bianchi significativi. I suoi costrutti linguistici e l'approccio orientato agli oggetti mirano ad aiutare i programmatori a scrivere codice chiaro e logico per progetti su piccola e grande scala. Python è tipizzato dinamicamente e raccolto in modo indesiderato. Supporta più paradigmi di programmazione, inclusa la programmazione strutturata (in particolare, procedurale), orientata agli oggetti e funzionale. Python è spesso descritto come un linguaggio "batterie incluse" a causa della sua libreria standard completa.

Android

Android è un sistema operativo mobile basato su una versione modificata del kernel Linux e altro software open source, progettato principalmente per dispositivi mobili touchscreen come smartphone e tablet. Android è sviluppato da un consorzio di sviluppatori noto come Open Handset Alliance e sponsorizzato commercialmente da Google. È stato presentato nel novembre 2007, con il primo dispositivo Android commerciale lanciato nel settembre 2008. È un software gratuito e open source; il suo codice sorgente è noto come Android Open Source Project (AOSP), che è principalmente concesso in licenza con la licenza Apache. Tuttavia, la maggior parte dei dispositivi Android viene fornita con software proprietario aggiuntivo preinstallato, in particolare Google Mobile Services (GMS) che include app principali come Google Chrome, la piattaforma di distribuzione digitale Google Play e la piattaforma di sviluppo Google Play Services associata. Circa il 70% degli smartphone Android gestisce l'ecosistema di Google; ecosistemi e fork di Android concorrenti includono Fire OS (sviluppato da Amazon) o LineageOS. Tuttavia, il nome e il logo "Android" sono marchi di Google che impongono standard per limitare l'uso del marchio Android ai dispositivi "non certificati" al di fuori del proprio ecosistema.

NODE JS

Node.js è un runtime JavaScript open source multipiattaforma orientato agli eventi per l'esecuzione di codice JavaScript, basato sul motore JavaScript V8 di Google Chrome. Molti dei suoi moduli principali sono scritti in JavaScript e gli sviluppatori possono scrivere nuovi moduli in JavaScript. JavaScript è stato originariamente utilizzato principalmente dal lato client. In questo scenario, gli script JavaScript, solitamente incorporati nell'HTML di una pagina Web, vengono interpretati da un motore di esecuzione incorporato direttamente in un browser. Node.js invece permette di utilizzare JavaScript anche per scrivere codice da eseguire lato server, ad esempio per produrre il contenuto di pagine web dinamiche prima che la pagina venga inviata al browser dell'utente. In questo modo Node.js consente di implementare il cosiddetto paradigma "JavaScript ovunque" (JavaScript ovunque), unificando lo sviluppo di applicazioni Web attorno ad un unico linguaggio di programmazione (JavaScript). NodeJs ha un'architettura orientata agli eventi che rende possibile l'I / O asincrono. Questo progetto mira a ottimizzare il throughput e la scalabilità nelle applicazioni web con molte operazioni di input / output, è ottimo anche per applicazioni web di sistema in tempo reale (ad esempio programmi di comunicazione in tempo reale o giochi per browser). Node.js è un progetto di sviluppo open source distribuito gestito dalla Node.js Foundation e facilitato dal programma di progetti collaborativi della Linux Foundation.
